# Interface-de-E-commerce-com-JS
Eu criei este projeto com o objetivo de estudar o funcionamento do localStorage e suas utilidades mais a fundo, e para aprimorar as minhas habilidades em JavaScript no geral.

Como usar: Para utilizá-lo, basta extrair o arquivo.zip e acessar o index.html.

Funcionalidades: Como dito anteriormente, o objetivo deste projeto é explorar as funcionalidades do localStorage, ao testar o site e atualizar a página, você verá que os itens no carrinho permanecem, a não ser que você limpe eles.

Observações: Em breve pretendo fazer uma versão mais atualizada e melhorada com mais funcionalidades, como adicionar produtos, por exemplo.
